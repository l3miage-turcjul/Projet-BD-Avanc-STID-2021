Le Projet de Base de données avancée


Le but de ce projet est d'appliquer les connaissances que nous avons acquises durant nos cours de gestion de bases de données et de bases de données avancées. Ici nous avons décidé d'utiliser comme données de base les vidéos youtubes qui ont été en tendances en France, c'est-à-dire les vidéos les plus populaires de youtubes en France. Cette activité, que nous avons travaillée en TP mais aussi depuis nos domiciles est une synthétisation de plusieurs compétences que nous avons acquises à STID

Les compétences que nous avons utiliser à l'occasion de ce projets sont :

  • La recherche et le netoyage de données
  • La création d'une base de données
  • L'analyse des données en fonctions d'une ou plusieurs composantes
  • La mise en place d'un site internet et de graphiques afin de synthétyser nos analyses
